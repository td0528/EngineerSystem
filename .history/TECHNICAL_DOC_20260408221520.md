# VONIKO 工厂集成管理系统 (Factory Management System) - 技术文档与更新日志

## 1. 系统概述 (System Overview)
在此记录系统架构与核心技术栈。
- **后端框架**: Python 3.11 + FastAPI
- **数据库**: SQLite (`engineering.db`, `voniko.db`) + SQLAlchemy (ORM)
- **前端技术**: 纯 HTML / CSS / JS (原生无框架，通过 FastAPI 挂载静态文件)
- **部署方案**: Docker + Docker Compose 或原生 Python 运行 (`uvicorn`)

## 2. 核心功能模块 (Core Features)
在此记录系统当前拥有的所有主要功能模块。

### 2.1 认证与用户管理 (Auth & Users)
- 提供用户登录与权限验证 (`/api/auth`)。
- 多角色支持（管理员、工程师、供应商等）。
- `users.html` 提供后台用户管理界面。
- **会话管理**: 采用 `beforeunload` + `unload` 事件机制，关闭标签页/浏览器时弹出确认对话框，确认离开后自动清除登录态，要求重新登录。
- **记住我**: 登录页「记住我」复选框可将账号密码持久保存，下次打开自动填充。

### 2.2 物料与库存管理 (Material & Warehouse Management)
- **物料管理** (`materials.html`, `/api/materials`): 支持物料的增删改查、图片上传与查看。具备高级图像查看器（支持拖拽平移、缩放等功能）。
- **统一说明记录**: 废弃原先单行的规格描述，改为多条说明记录组成的列表（`MaterialNote`模型），支持动态的新增、编辑、删除。
- **SAP号码自动生成**: 新建物料时自动生成100000-999999范围内的下一个连续 SAP号码（`GET /api/materials/next-sap-code`）。
- **批量操作**: 支持通过复选框批量选择物料并一键删除（`POST /api/materials/batch-delete`）。
- **仓库管理** (`warehouse.html`, `/api/warehouse`): 入库、出库操作，并记录库存变动。
- **历史全链路追溯**: 在库存一览中支持多选物料，置顶悬浮对比。通过“历史溯源”面板展示从供应商确认、发货、到仓库收货、退货、再补货及出库的全生命周期时间轴。
- **库存预警** (`stock-alerts.html`): 根据设定的安全库存提供库存不足警报。
- **金额标准化**: 全站数字展示统一千分位格式，单价支持 **VND/USD** 货币实时切换与存储。

### 2.3 采购与订单管理 (Orders Management)
- **采购订单** (`orders.html`, `/api/orders`): 整合采购需求、生成订单，记录供应商、物料、数量及交期等。
- **作用说明联动**: 订单明细的作用说明现在与物料管理的说明记录完全联动，可直接下拉选择历史说明或动态新增。
- **数据导入导出**: 支持从Excel/CSV批量导入订单数据（`POST /api/orders/import`），以及导出当前数据为Excel。
- **批量操作**: 支持通过复选框批量选择订单明细并一键删除（`POST /api/orders/items/batch-delete`）。
- **快速新建物料**: 在创建订单时可直接快速新建物料，SAP号码和规格说明均可一并保存（说明将自动转存为第一条说明记录）。
- 供应商可以通过门户直接查看相关订单。

### 2.4 供应商门户 (Supplier Portal & Admin)
- **供应商接口** (`supplier-portal.html`, `/api/supplier`): 专供外部供应商登录的独立界面，使用自身 Token 身份访问数据。
- **表格高级功能**: 支持表头列级别的具体搜索与排序。
- **供应商管理** (`supplier-admin.html`): 集成标准侧边栏布局，位于「采购与库存」分类下，配备供应商下拉选择器。
- **供应商搜索** (`GET /api/orders/suppliers/search`): 支持空字符串搜索返回全部供应商，默认上限 50 条。

### 2.5 生产与设备监控 (Production & Devices)
- **设备监控面板** (`devices.html`, `/api/devices`): 实时展示工厂设备的运行状态和相关监测数据点。

### 2.6 其他辅助模块 (Others)
- **任务系统** (`tasks.html`, `/api/tasks`): 内部流程任务分配与跟踪。
- **资料与文档** (`documents.html`, `/api/documents`): 图纸、规范等工艺文档的上传、分类与下载。
- **操作记录** (`records.html`, `/api/records`): 记录各类出入库、系统操作的日志记录。
- **系统配置** (`configurations.py`, `/api/config`): 动态管理系统全局参数。
